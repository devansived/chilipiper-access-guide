How to use this guide

For IT / provisioning:
  1. Select the new hire's Okta team from the dropdown.
  2. Review all rows marked ✅.
  3. Provision those ChiliPiper licenses and workspaces.
  4. Only grant ⛔ items by exception.

To update this guide, refresh the underlying CSV export, regenerate the HTML, and redeploy via GitHub Pages.
